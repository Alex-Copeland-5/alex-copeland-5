Resources

A curated list of tools, links, and guides I recommend.

A few resources I recommend for mathematics or starting out with programming!

  • Khan Academy - Free online courses on various subjects including math and programming.
  • Codecademy - Interactive platform to learn coding in different programming languages.
  • Stack Overflow - A community-driven Q&A site for programmers to find solutions to coding problems.
  • Math is Fun - A website that makes learning math enjoyable with clear explanations and interactive activities.
  • GeeksforGeeks - A platform for learning computer science concepts and programming through articles and coding challenges.
  • W3 Schools - A comprehensive resource for learning web development technologies including HTML, CSS, and JavaScript.
  • Wolfram Alpha - A computational search engine that provides answers to factual queries by computing answers from externally sourced data.
